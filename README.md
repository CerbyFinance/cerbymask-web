# Getting Started 

## Install dependencies

After cloning the project, navigate to the root folder and run:

```bash
$ npm install
```

## Build


In order to build the webpage, execute:

```bash
$ npm run build
```

## Serve

In order to serve a reloadable webpage:

```bash
$ npm run start
```
